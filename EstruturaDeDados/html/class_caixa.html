Documentação das enumerações

enum Caixa::Eficiencia

Representa a eficiencia de um caixa, sendo que esta afeta o tempo para se atender um cliente da seguinte forma:
Tempo para atender cliente = quantidade de produtos multiplicado pelo tempo de digitação de cada item (em segundos) + tempo para receber o cheque (em segundos)

Valores da enumeração
eficiente 

Tempo de digitação p/ produto = 1s , Tempo para receber cheque = 10s

medio 

Tempo de digitação p/ produto = 2s , Tempo para receber cheque = 25

ruim 

Tempo de digitação p/ produto = 4s , Tempo para receber cheque = 60s
